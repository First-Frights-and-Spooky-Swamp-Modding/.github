# First Frights and Spooky Swamp Modding
We are an organisation focused on developing tools and documentation for Scooby-Doo! First Frights and Scooby-Doo! and the Spooky Swamp. Our goal is to make tools for modding First Frights and Spooky Swamp. As well as better understand how the games work under the hood.
